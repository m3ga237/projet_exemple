# projet_exemple
exemple de petit projet web

ce projet est un projet que je me suis imposé en vue de le partager pour montrer mes compétences actuelles en dévellopement web en vue d'un stage

le sujet a été imposé par un IA générative 
et réalisé en 2 à 3h

afin de tester l'app vous pouver trouver les identifiant et mdp des compte du site dans la table user de la bdd ou dans le script driectement 

ce site contient diverses page permettant après connexion la gestion d'un inventaire 
ajout d'objet / supression / modification

# important 
pensez bien a importer le script en .sql fourni sur un serveur WAMP ou LAMP local afin de pouvoir faire fonctionner le site
